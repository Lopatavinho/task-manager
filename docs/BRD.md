# Documento de Requisitos de Negócio (BRD)

## 1. Propósito
Desenvolver um sistema simples de gerenciamento de tarefas que permita que usuários criem, visualizem, atualizem e excluam tarefas, com autenticação segura.

## 2. Objetivos de Negócio
- Organizar tarefas pessoais e profissionais.
- Permitir login seguro e individual.
- Prover acesso rápido às tarefas por status.

## 3. Público-Alvo
Usuários individuais que desejam gerenciar suas atividades diárias.

## 4. Regras de Negócio
- Cada usuário só pode ver suas próprias tarefas.
- O título da tarefa é obrigatório.
- Status pode ser “Pendente”, “Em andamento” ou “Concluída”.
- O sistema deve registrar data de criação e atualização.
